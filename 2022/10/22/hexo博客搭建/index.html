hexo博客搭建

搭建前的准备

下载Git和Node.js

安装hexo

1.进入cmd窗口输入指令：npm install -g hexo-cli

2.创建一个文件夹myblog，然后cd到这个文件夹下（或者在这个文件夹下直接右键git bash打开）。

初始化hexo

hexo init

新建完成后，指定文件夹目录下有：
node_modules: 依赖包
public：存放生成的页面
scaffolds：生成文章的一些模板
source：用来存放你的文章
themes：主题
_config.yml: 博客的配置文件

http://localhost:4000/是初始界面，我们需要部署到github上。

将Hexo部署到Github

1.Github创建个人仓库
创建一个和用户名相同的仓库，后面加.github.io，

2.生成ssh添加到Github
在Github上创建仓库完成之后，需要设置ssh免密登录
在cmd窗口执行：

git config –global user.name “yourname”
git config –global user.email “youremail”

3.进入到家目录，右击打开git bash 。
输入：ssh-keygen -t rsa -C qq号@qq.com，然后敲三次回车，

3.接着就会发现家目录下多了一个.ssh目录，打开后有一个公钥，一个私钥。
id_rsa.pub是公钥，我们需要打开复制里面的内容。

然后进入github，在setting里面进行操作

进行以下操作

发现我们需要一个密钥，把我们刚刚复制的密钥粘进去，title随便起

点击 Add SSH Key

3.进行部署
打开站点配置文件 _config.yml，翻到最后，修改为GitHub账户名

1.修改配置文件

修改内容：

deploy:
type: git
repo: git@github.com:goubin18/goubin18.github.io.git
branch: main

注意：后面有空格

repo：获取步骤如下

点进自己刚刚创建的仓库，复制路径

2.找到自己的博客路径打开

这个时候需要先安装deploy-git ，也就是部署的命令,这样你才能用命令部署到GitHub。

npm install hexo-deployer-git –save

2.然后依次执行以下命令：

hexo c #清除缓存文件 db.json 和已生成的静态文件 public
hexo g #生成网站静态文件到默认设置的 public 文件夹(hexo generate 的缩写)
hexo d #自动生成网站静态文件，并部署到设定的仓库(hexo deploy 的缩写)
deploy时输yes

然后就可以使用xxx.github.io来访问博客
